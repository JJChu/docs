# 前端进阶路线

这里先以阿里职级体系为例来说明各个阶层的技术要求：

  • P4：前端基础编程能力；
  • P5：前端基础技能；
  • P6：组件化、工具、质量监控；
  • P7：组件体系、工具链、持续集成；
  • P8：前端体系；

然后再来说说前端的进阶路线，包括需要了解的技能和知识点：

# 1. 建立前端知识架构

  • 高效的学习方法：“知识整理分类”和“追本溯源”；
  • 前端知识图谱

# 2. JS 语言

  • JS 语言基础和其运行环境
  • JS 文法、语义、运行时
  • JS 的核心功能和设计思路

# 3. 浏览器的工作原理和 API

  • 理解浏览器的整个工作过程
  • DOM BOM XHR 等浏览器核心 API

# 4. 编程与算法

  • 训练与语言环境无关的基础编程实例
    • TicTacToe/井字棋（递归）
    • 寻路问题（搜索）
    • 出现最多的50个字符串（哈希树/排/结构）
    • 判断点击区域（计算几何）
    • 括号匹配（栈）
    • wildcard（贪心）
  • 训练与 JS 和 Web 相关的变成实例
    • Promise 与异步编程
    • 正则表达式和文本处理
    • Proxy 与双向绑定
    • 使用 Range 实现 DOM 精确操作
    • 使用 CSSDOM 实现视觉交互
    • 解析一个四则运算表达式

# 5. 组件化

  • 理解组件化的基本概念、应用场景和基本思路；
  • 了解组件化的历史与现状，提升对组件化这一工程手段的认知
    • 搭建自己的组件库（elementUI）

# 6. 工具链

  • 实现一套可以用于实际开发的工具链模型；
    • 目录结构和初始化工具（node）
    • 设计并实现一个构建工具（webpack rollup babel...）
    • 设计并实现一个调试工具（）
    • 设计并实现一个单元测试工具（mocha）

# 7. 发布和持续集成

  • 实现一个持续继承和前端发布体系
    • 实现一个 web 服务（node express koa）
    • 实现一个发布系统（git）
  • 学习如何用持续集成保障前端代码质量
    • 如何使用 Git Hook
    • 如何使用 Lint 工具（eslint）
    • 如何使用无头浏览器（PhantomJS、puppeteer）
    • 如何实现 DOM 树的检查
